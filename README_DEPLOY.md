# 🚀 Deployment Guide for LITTLE JALABIYA

If your photos are not showing up, please follow these critical steps:

## 1. For Google Sites Users (Important)
Google Sites "Embed code" blocks local assets. You cannot just paste the code if you want images to show.
**The Fix:**
- Create a [GitHub](https://github.com) repository and upload all files (including the `assets` folder).
- Turn on **GitHub Pages** in the repository **Settings → Pages**.
- Instead of using "Embed Code" in Google Sites, use the **"Embed by URL"** tab.
- Paste your live GitHub link (e.g., `https://your-username.github.io/your-project/`).
- This will load the site perfectly with all photos visible!

## 2. Using GitHub / Netlify (Recommended)
- Upload the entire folder.
- Ensure `index.html` is in the main (root) folder.
- Everything will work automatically.

## 3. Playing it Locally (Offline)
- **CRITICAL**: You MUST **Extract All** from the ZIP before opening `index.html`. 
- If you open it while still inside the zip, the images will be blocked by your browser.
- Once extracted, simply open `index.html` and everything will be visible!
